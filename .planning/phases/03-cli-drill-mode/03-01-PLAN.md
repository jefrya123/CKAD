  <name>Task 1: Implement lib/session.sh with session JSON management</name>
  <files>lib/session.sh, test/unit/session.bats</files>
  <behavior>
    - session_write creates valid JSON at CKAD_SESSION_FILE with mode, scenario_id, scenario_file, namespace, started_at, time_limit, end_at fields
    - session_write creates CKAD_CONFIG_DIR if it does not exist (mkdir -p)
    - session_read populates SESSION_MODE, SESSION_SCENARIO_ID, SESSION_SCENARIO_FILE, SESSION_NAMESPACE, SESSION_STARTED_AT, SESSION_TIME_LIMIT, SESSION_END_AT globals
    - session_read returns EXIT_NO_SESSION (3) when session file is missing
    - session_clear removes the session file
    - session_require calls session_read; prints error and exits EXIT_NO_SESSION if no session
    - session_write stores end_at as epoch seconds (started_epoch + time_limit) for cross-platform date portability
  </behavior>
  <action>
    Write tests first in test/unit/session.bats following existing test patterns (absolute path loading, bash -c subshells for isolation, no test-helper.bash). Test each function: write creates valid JSON (verify with jq), read populates SESSION_* globals, read returns 3 on missing file, clear removes file, require exits on missing session.

    Then implement lib/session.sh:
    - Header: `# shellcheck shell=bash` + `# shellcheck disable=SC2034` (SESSION_* globals used by callers)
    - session_write MODE SCENARIO_ID SCENARIO_FILE NAMESPACE TIME_LIMIT: compute started_at via `date -u +"%Y-%m-%dT%H:%M:%SZ"`, end_at via `$(( $(date +%s) + time_limit ))`, write JSON with `jq -n --arg/--argjson`, mkdir -p CKAD_CONFIG_DIR before write
    - session_read: use `jq -r '.field // empty'` for each field, set SESSION_* globals, return EXIT_NO_SESSION if file missing
    - session_clear: rm -f CKAD_SESSION_FILE
    - session_require: call session_read; on failure print error "No active drill session. Run 'ckad-drill drill' first." and exit EXIT_NO_SESSION

    IMPORTANT: Use `jq` for all JSON operations (not yq). Use `((n++)) || true` pattern for arithmetic with set -e. Store epoch seconds for end_at (not ISO string) per research doc cross-platform recommendation.
  </action>
  <verify>
    <automated>cd /home/jeff/Projects/cka && bats test/unit/session.bats</automated>
  </verify>
  <done>All session.bats tests pass. session_write creates valid JSON, session_read populates globals, session_clear removes file, session_require exits on missing session.</done>
